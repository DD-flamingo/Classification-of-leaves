# Classification of Leaves

## Project Description
This project creates a web application that identifies a leaf from an image scanned or uploaded by the user.

## Features
- Image upload
- Leaf classification using ML/DL
- Web-based interface

## Technologies Used
- Python
- TensorFlow / PyTorch
- OpenCV
- Streamlit / Flask

## How to Run
1. Install required libraries
2. Run the application
3. Upload a leaf image

## Future Work
- Improve accuracy
- Add more species
- GIS-based tree location mapping
